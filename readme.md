# Éditeur de Bandelettes A4

Un outil web simple et efficace pour créer et imprimer des étiquettes, bandelettes ou fiches au format A4 avec différents types de mise en page.

## 🚀 Fonctionnalités

- **Plusieurs mises en page :**
  - Pleine page
  - Tiers (Horizontal)
  - Moitiés (Vertical)
  - 8 Rectangles
  - 16 Rectangles
- **Adaptation automatique du texte :** La taille de la police s'ajuste automatiquement pour remplir l'espace disponible de manière optimale.
- **Édition en direct :** Cliquez et modifiez le texte directement sur la page.
- **Sélection de polices :** Choisissez parmi une sélection de polices (Caveat Brush, Sarabun, Raleway, Arial, etc.).
- **Importation de liste :** Importez une liste d'éléments (un par ligne) pour générer automatiquement plusieurs étiquettes.
- **Export & Impression :** Imprimez directement ou enregistrez au format PDF via les fonctions natives du navigateur.

## 🛠️ Installation

Aucune installation n'est requise. Ce projet est composé uniquement de fichiers statiques (HTML, CSS, JS).

1. Clonez ou téléchargez le dépôt.
2. Ouvrez le fichier `index.html` dans votre navigateur web préféré.

## 📂 Structure du projet

- `index.html` : Structure de l'application et de la barre d'outils.
- `style.css` : Styles de l'interface et gestion spécifique de la mise en page pour l'impression.
- `script.js` : Logique de manipulation du DOM, gestion du layout et algorithme d'ajustement automatique du texte.

## 🎨 Utilisation

1. **Choisir une mise en page** via la barre d'outils en haut.
2. **Modifier le texte** en cliquant directement dans les zones éditables.
3. **Changer la police** pour personnaliser l'apparence.
4. **Importer une liste** si vous avez beaucoup d'étiquettes à créer d'un coup.
5. **Cliquer sur Imprimer** ou **Enregistrer en PDF**.
